# go-family-finance
family finance
Development Environment Setup:
1. Ensure Docker or Podman is correctly installed on the system.
2. Set the current directory to `go-family-finance/deployments/dev`.
3. Run `docker-compose up -d` and wait for the essential development environment components like the PG database, Redis, etc., to start correctly.
4. Let's Go.


开发环境搭建：
1、确保系统中正确安装Docker或者Podman
2、将当前目录设置为 go-family-finance/deployments/dev
3、运行docker-compose up -d 等待开发环境必须的PG数据库，Redis等正确运行
4、Let's Go
